# Менеджер списка дел

## Описание проекта

Веб-приложение для управления списком задач.

Пользователь может:

* добавлять задачи;
* просматривать список задач;
* отмечать задачи как выполненные;
* удалять задачи;
* сохранять данные между запусками приложения.

## Используемые технологии

* Python 3
* Flask
* HTML
* CSS
* JSON

## Установка и запуск

1. Установить Python.
2. Установить Flask:

pip install -r requirements.txt

3. Запустить приложение:

python app.py

4. Открыть браузер:

http://127.0.0.1:5000
