Тестирование приложения

Тест 1. Добавление задачи
Действие: пользователь вводит название задачи и нажимает кнопку «Добавить».
Ожидаемый результат: задача появляется в списке.
Результат: успешно.

Тест 2. Изменение статуса задачи
Действие: пользователь нажимает кнопку «✓».
Ожидаемый результат: задача отмечается как выполненная.
Результат: успешно.

Тест 3. Удаление задачи
Действие: пользователь нажимает кнопку «🗑».
Ожидаемый результат: задача удаляется из списка.
Результат: успешно.

Тест 4. Сохранение данных
Действие: приложение закрывается и запускается повторно.
Ожидаемый результат: ранее добавленные задачи сохраняются.
Результат: успешно.
