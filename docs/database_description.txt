Для хранения данных в приложении используется файл tasks.json в формате JSON.

Каждая задача представляет собой объект со следующими полями:

* id — уникальный идентификатор задачи;
* title — название задачи;
* completed — статус выполнения (true/false).

Пример записи:

{
"id": 1,
"title": "Сделать лабораторную работу",
"completed": false
}

При запуске приложения данные загружаются из файла tasks.json.
После добавления, изменения или удаления задачи файл автоматически обновляется.
